# Digital Evidence Management System

## Project Overview

Digital Evidence Management System is a C++ application developed for secure storage, management, and verification of digital evidence. The system supports role-based access control and efficient evidence handling using advanced data structures.

---

## Technologies Used

- C++
- Object-Oriented Programming
- Data Structures
- Windows CryptoAPI

---

## Features

- Evidence storage and retrieval
- SHA-256 file integrity verification
- Role-Based Access Control (RBAC)
- User authentication system
- Audit logging
- Multi-criteria search functionality
- Evidence filtering and sorting
- Report generation

---

## User Roles

- Admin
- Investigator
- Analyst
- Auditor
